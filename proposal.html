Pokemon Tracker

RESTful API

PokéApi: The RESTful Pokémon API

Data

Data from this API will be used to populate the website will all the info about Pokemon there is to exist.

We will track users of the site, and collect information about their favorite Pokemon, games they've played, and more.

Database Entities

Users -> (UserName, Password, Email)

FavoritePokemons -> (UserName, PokemonName)

PlayedGames -> (UserName, GameName)

Users

Pokemon fans and enthusiasts

Goals

Provide a one-stop shop for all Pokemon related information

Provide a platform for Pokemon fans to keep track of their own personal Pokemon-related information

Motivation

The reason we decided on this topic is because both of us are big Pokemon fans and thought it would be a lot of fun to use the PokeAPI
